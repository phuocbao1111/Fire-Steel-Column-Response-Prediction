# Fire-Steel-Column-Response-Prediction
This tool is developed as part of the research project "Application of hybrid machine learning techniques for evaluating the ultimate strength of steel columns under axial compression". It provides a robust and efficient way to predict the force-displacement behavior of I-section steel columns by leveraging a trained XGBoost surrogate model.
1. Download all files from this repository to your local computer.
2. Install the required libraries in Python by running this command in your terminal:
      pip install streamlit xgboost pandas numpy matplotlib joblib shap
3. Open your Command Prompt (CMD) or Terminal and navigate to the project directory where you saved the files.
4. Execute the command: streamlit run GUI.py.
5. Once the command is executed, your web browser will automatically open the interface.
